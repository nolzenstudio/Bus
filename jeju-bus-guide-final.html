위치 권한이 꺼져 있어요

현재 기본 위치(제주시청) 기준으로 정류장을 보여드리고 있어요. 내 주변 정류장을 보려면 아래 방법으로 위치 권한을 켜주세요.
  1. 설정 → 개인정보 보호 및 보안 → 위치 서비스 켜기
  2. 아래로 스크롤 → Safari 웹사이트 → "앱을 사용하는 동안"
  3. 이 페이지로 돌아와 새로고침 후 아래 "다시 시도"
  1. 주소창 왼쪽 자물쇠 🔒 아이콘 → 권한 → 위치 → 허용
  2. 안 보이면 Chrome ⋮ → 설정 → 사이트 설정 → 위치에서 허용
  3. 폰의 위치(GPS)도 켜져 있어야 해요 → 아래 "다시 시도"
주소가 https:// 로 시작하는지 확인하세요. http로 접속하면 위치 권한이 아예 뜨지 않습니다.
